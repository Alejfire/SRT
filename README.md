# Sistema de Matchmaking en Tiempo Real

## Descripción
Este proyecto implementa un sistema de matchmaking competitivo utilizando conceptos de programación en tiempo real.

## Características
- Concurrencia con hilos
- Uso de semáforos para sincronización
- Emparejamiento de jugadores
- Registro de partidas en archivo

## Cómo ejecutar
```bash
python matchmaking.py
